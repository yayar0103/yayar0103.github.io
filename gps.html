GPS

使用waveshare公司GPS產品，型號L76X GPS HAT

至waveshare網站下載產品 使用手冊 與 Demo code
手冊內有多種測試GPS產品是否可以運作方法:
可以根據使用不同方法，接收GPS衛星回傳資料。

特別注意

1.黃色扣環需要根據使用不同方法，調整不同方案:
使用方法1用A；
使用方法3直接將GPS腳位對接樹莓派腳位用B；
使用方法3直接將GPS差在樹莓派USB插槽用A。
2.使用方法3時，根據使用不A或B調整串口裝置程式:
使用B方案，程式用[/dev/ttyS0]；
使用A方案，程式用[/dev/ttyUSB0]
  1. 1.直接在電腦上測試(用USB連接電腦)

    2. 開啟電腦上 [串口調適助手] 內建裝置，將[串口號]選擇為連接GPS產品的串口。[串列傳輸速率]為9600，[資料位]為8...其他不用改，選擇好後按下[打開串口]，就能接收到資料。

  2. 2.Arduino測試

    3. 不討論，自行參閱waveshare公司使用手冊

  3. 3.樹梅派上測試

    4. 1. 下載Demo code 至樹莓派上，跟著使用手冊P10-P12的步驟執行，當執行完 [minicom -D /dev/ttyS0 -b 9600] 時(ttyS0請根據上面特別注意更改)，有出現下圖紅色區域，表是成功讀取GPS衛星資料。
    2. 繼續執行使用手冊P12，RUNNING CODES部分，先更改資料夾中config.py檔案，根據特別注意第二點更改檔案中tty，再直接執行此兩步驟，為解析GPS衛星回傳資料(將紅色區塊資料解讀為黃色區塊)

    3. 出現下圖黃色區塊表示解讀成功。

  4. 4.STM32測試

    5. 不討論，自行參閱waveshare公司使用手冊
